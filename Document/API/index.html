基本概念

API (Application Program Interface)，應用程式的接口，軟體串接的橋樑

網站可以使用 Web APIs 透過 HTTP 協議的方式，向第三方公開的 URL 端發出需求，獲取對方提供的 JSON

Action HTTP Method Description
Create POST Creates a new resource
Read GET Retrieves a resource
Update PUT / PATCH Updates an exiting resource
Delete DELETE Deletes a resource

操作

  1. 閱讀 API 文件
  2. 取得端點：URL
  3. 使用 HTTP Request 獲取資料
閱讀 API 文件

範例：Studio Ghibli API Document

取得端點

範例文件裡頭， films 部分，右側有其 API 端點（URL)

使用 HTTP Request 獲取資料
  1. 新建 js 檔案，新增 XML HttpRequest 物件 
    var request = new XMLHttpRequest()
  2. 使用 open() 函式連接 
    request.open('GET', 'https://ghibliapi.herokuapp.com/films')
  3. 得到權限後（此範例無需 key )，使用 onload() 函式，進入資料 
    request.onload = function () {}
  4. 最後使用 send() ，送出邀請
    request.send()

每次 HTTP 發出 Request，就會回傳 Status Code

Status Code Description
Success 200 請求成功
Not found 404 請求失敗 
if (request.status >= 200 && request.status < 400) {
        data.forEach(movie => {
        console.log(movie.title)
    })
} else {
    console.log('error')
}

需要處理 HTTP 回傳的資料，使用 JSON.parse() 將 JSON 轉換為 Javascript 物件，再對資料做處理 

var data = JSON.parse(this.response)
